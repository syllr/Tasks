# Tasks - IntelliJ IDEA 任务管理插件

一个运行在 JetBrains IDE 2025.3 上的轻量级任务管理工具，在 IDE 侧边栏直接管理你的开发任务。

## 功能特性

- 📝 **任务管理**：支持添加、编辑、删除任务
- 🔄 **进度跟踪**：三种任务状态循环切换 - 待办 → 进行中 → 完成
- 📊 **统计概览**：顶部显示各状态任务数量统计
- 💾 **两级存储**：
  - **项目级别**：任务存储在项目 `.idea/` 目录下，跟随项目一起版本控制（可选）
  - **用户级别**：任务存储在用户目录 `~/.todoTasks/`，只在本地存在不影响项目
- 🎨 **原生界面**：使用 JetBrains 原生 UI 组件，完美适配 IDE 主题
- 🚀 **轻量简洁**：无第三方依赖，启动飞快，不影响开发效率

## 界面布局

```
┌─────────────────────────────────────────────┐
│  待办: 2 | 进行中: 1 | 完成: 3               │  ← 顶部统计栏
├─────────────────────────────────────────────┤
│  ┌───────────────────────────────────────┐  │
│  │  任务标题                  [状态][编辑][删除] │  ← 任务项
│  │  任务描述                                   │  ← 左侧：标题+描述
│  └───────────────────────────────────────┘  │  ← 右侧：三个按钮水平居中
│  ┌───────────────────────────────────────┐  │
│  │  另一个任务                [状态][编辑][删除] │  │
│  └───────────────────────────────────────┘  │
│                                             │
│                 [ + 添加任务 ]               │  ← 底部添加按钮
└─────────────────────────────────────────────┘
```

## 构建安装

### 环境要求

- JDK 17+
- Gradle 8+

### 构建插件包

```bash
./gradlew buildPlugin
```

构建产物在 `build/distributions/Tasks-1.0.0.zip`，可以直接在 IDE 中从磁盘安装插件。

### 调试运行

```bash
./gradlew runIde
```

会自动启动一个新的 IDE 实例加载插件进行调试。

## 存储位置

- **项目级别任务**：`[项目根目录]/.idea/tasks.json`
- **用户级别任务**：`~/.todoTasks/[项目名称].json`

两种存储可以在新建/编辑任务时自由选择。

## 技术栈

- **语言**：Kotlin
**框架**：IntelliJ Platform 2025.3
**序列化**：Gson（JSON 存储）
**构建**：Gradle with IntelliJ Platform Plugin 2.x
