# Tasks - IntelliJ IDEA 任务管理插件

一个运行在 JetBrains IDE 2025.3 上的轻量级任务管理工具，在 IDE 侧边栏直接管理你的开发任务，无需离开IDE就能跟踪工作进度。

## 功能特性

- 📝 **任务管理**：支持添加、编辑、删除任务
- 🔄 **进度跟踪**：三种任务状态循环切换 - 待办 → 进行中 → 完成
- 👆 **点击查看详情**：点击任务卡片弹出弹窗显示完整标题、描述、创建时间
- 🔍 **状态筛选**：顶部左侧下拉框按状态筛选任务（全部/待办/进行中/完成）
- 📊 **统计概览**：顶部右侧显示各状态任务数量统计
- ✅ **输入校验**：新建/编辑任务时禁止空标题
- ⏱️ **智能排序**：任务按创建时间倒序排列，最新任务永远在最上面
- 💾 **两级存储**：
  - **项目级别**：任务存储在项目 `.idea/` 目录下，跟随项目一起版本控制（可选）
  - **用户级别**：所有用户级任务统一存储在 `~/.todoTasks/tasks.json`，只在本地存在不影响项目
- 🎨 **原生界面**：使用 JetBrains 原生 UI 组件，完美适配 IDE 浅色/深色主题
- 🎯 **圆角卡片设计**：每个任务独立圆角边框卡片，清晰干净
- 🚀 **轻量简洁**：无多余依赖，启动飞快，不影响开发效率

## 构建安装

### 环境要求

- JDK 17+
- Gradle 8+

### 构建插件包

```bash
./gradlew buildPlugin
```

构建产物在 `build/distributions/Tasks-<version>.zip`，可以直接在 IDE 中通过 `Settings → Plugins → Install plugin from disk` 安装。

### 调试运行

```bash
./gradlew runIde
```

会自动启动一个新的 IDE 实例加载插件进行调试。

## 存储位置

- **项目级别任务**：`[项目根目录]/.idea/tasks.json`
- **用户级别任务**：`~/.todoTasks/tasks.json`（所有项目的用户级任务都存在这里）

两种存储可以在新建/编辑任务时自由选择。

## 技术栈

- **语言**：Kotlin
- **框架**：IntelliJ Platform 2025.3
- **序列化**：Gson（JSON 存储）
- **构建**：Gradle with IntelliJ Platform Plugin 2.x
